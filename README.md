# Project Dependencies

This project relies on the following Python libraries and tools for proper functionality. Please install them before proceeding with the development or testing.

---

## Core Libraries
- **numpy**: For numerical computations.
- **opencv-python**: For video and image processing.
- **mediapipe**: For face detection and tracking.
- **scipy**: For signal processing and filtering.
- **matplotlib**: For plotting and data visualization.

---

## Additional Libraries (Optional)
- **scikit-learn**: For model evaluation and performance analysis.
- **pandas**: For data manipulation and storage if needed in extended workflows.

---

## Installation
Use the following command to install all dependencies at once:
```bash
pip install numpy opencv-python mediapipe scipy matplotlib scikit-learn pandas
